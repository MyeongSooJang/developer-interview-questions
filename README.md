# 개발자 면접 질문 리스트

Java 백엔드 개발자 취업 면접을 준비하기 위한 질문 저장소입니다.


## Categories

### 기본 CS

- 자료구조
- 알고리즘
- 운영체제
- 컴퓨터 구조

### 언어 및 프레임워크

- Java
- Spring
- Spring Boot
- JPA
- Testing
- [Design Pattern - Singleton](language-framework/design-pattern/singleton.md)

### 데이터베이스

- RDBMS
- 트랜잭션
- 인덱스
- SQL
- Redis

### 네트워크 및 아키텍처

- 네트워크 기본
- HTTP
- REST API
- 인증과 인가
- 서버 아키텍처

### 시스템 설계

- 확장성
- 캐싱
- 메시지 큐
- 동시성
- 설계 연습

## Question Template

```md
## Q. 질문 내용

### 답변
면접에서 실제로 말할 답변을 문장 형태로 정리합니다.

### 꼬리질문
- 이어서 나올 수 있는 질문을 정리합니다.

### 키워드
- 반드시 언급해야 할 키워드
```

## Suggested Review Order

1. Java, Spring, JPA를 먼저 정리합니다.
2. 트랜잭션, 인덱스, HTTP, REST API를 함께 복습합니다.
3. 운영체제, 자료구조, 알고리즘 등 기본 CS를 보강합니다.
4. 캐싱, 동시성, 메시지 큐, 확장성 설계로 시스템 설계 질문을 준비합니다.

## Writing Rules

- 질문 제목은 `## Q.`로 시작합니다.
- 답변은 면접에서 실제로 말할 수 있는 문장 형태로 적습니다.
- 꼬리질문은 실제 면접에서 이어질 수 있는 질문 위주로 적습니다.
- 키워드는 면접 답변에서 반드시 말해야 하는 단어만 남깁니다.
- 실제로 찾아본 면접 질문이 있을 때만 관련 파일을 새로 만들고 정리합니다.
