## Question Template

```md
## Q. 질문 내용

### 답변
면접에서 실제로 말할 답변을 문장 형태로 정리합니다.

### 키워드
- 반드시 언급해야 할 키워드

## Q. 이어질 수 있는 추가 질문

### 답변
짧게 답변합니다.
```

## Writing Rules

- 질문 제목은 `## Q.`로 시작합니다.
- 메인 질문은 `답변`과 `키워드`를 함께 정리합니다.
- 답변은 면접에서 실제로 말할 수 있는 문장 형태로 적고, 너무 길게 늘리지 않습니다.
- 이어질 수 있는 추가 질문은 별도 `## Q.`로 만들고 `답변`만 짧게 적습니다.
- 키워드는 메인 질문에서 반드시 말해야 하는 단어만 남깁니다.
- 실제로 찾아본 면접 질문이 있을 때만 관련 파일을 새로 만들고 정리합니다.
